Fractal Tree

The tree that you saw first on index page was made with turtle. Turtle is a graphical module that comes built in with python programming language. Below is the source code for the tree.


import turtle

hr = turtle.Turtle()
hr.color("white", "white")

ts = turtle.getscreen()
ts.bgcolor("black")

hr.left(90)
hr.speed(1500)

def tree(i):
    if i < 4:
        return
    else:
        hr.forward(i)
        hr.left(30)
        tree(3 * i/4)
        hr.right(60)
        tree(3 * i/4)
        hr.left(30)
        hr.backward(i)
tree(100)
turtle.done()
